MyFitMomAndDad

A nutrition tracking application focused on Indian dietary habits with distinct Admin and User roles.

Users can search and view nutritional information for various food items in multiple languages (English, Hindi, Marathi).
Admins manage food item data, update nutrition values, and generate reports through a dedicated dashboard.
Technology Used: Spring Boot, Java, Angular, Ionic, HTML, CSS, SQL.

Plant Match

A deep learning model designed to predict crop names from image-based data using convolutional neural networks.

Included image preprocessing, augmentation, and model training for improved accuracy.
Technology Used: Python, TensorFlow, CNN, OpenCV.

Next Word Prediction

A text generation project using LSTM-based sequential modeling to predict the next word in a sentence.

Trained an LSTM model to learn word patterns and generate next-word predictions.
Technology Used: Python, Keras, LSTM, NLP.
